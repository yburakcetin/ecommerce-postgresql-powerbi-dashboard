# Power BI Dashboard

This folder contains the completed Power BI dashboard deliverables for the ecommerce analytics portfolio project.

## Files

| File | Description |
|---|---|
| `ecommerce_dashboard.pbix` | Final Power BI dashboard file |
| `dashboard_overview.png` | Screenshot of Page 1: Executive Overview |
| `dashboard_product_customer.png` | Screenshot of Page 2: Product & Customer Analysis |
| `dashboard_revenue_adjustments_shipping.png` | Screenshot of Page 3: Revenue Adjustments & Shipping |
| `ecommerce_dashboard.pdf` | PDF export of the dashboard |
| `powerbi_measures.dax` | DAX measures used for dashboard KPIs |
| `powerbi_theme.json` | Power BI theme used for consistent colors and fonts |

## Dashboard Pages

1. Executive Overview
2. Product & Customer Analysis
3. Revenue Adjustments & Shipping

## Main KPIs

- Total Revenue
- Gross Revenue
- Total Orders
- Average Order Value
- Total Customers
- Total Units Sold
- Revenue per Customer
- Discounted Order %
- Estimated Discount Amount
- Average Shipping Fee

## Dashboard Purpose

The dashboard helps business users understand ecommerce sales performance, product and customer revenue drivers, country-level performance, discount impact, and shipping fee patterns.
